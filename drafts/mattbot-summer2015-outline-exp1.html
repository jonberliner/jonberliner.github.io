By Jon Berliner

Paper: Using GPs to model human reward-based decision problems with infinite choice sets

Motivations

Major

  • normative model for reward-based decision making with an infinite choice set
  • previous function learning centered on learning ground truth (e.g. Griffiths 2009)
  • actually gives us a normative parameterization of the GP to study, which is not true for using GPs to infer functions generated from different familiesm which is how previous studies work
  • our approach of generating stimuli with GPs rather than from families of functions allows us to systematically study GPs' effectiveness as inference machines. (maybe the same as above?)

Minor

  • shows direct influence of structure independent of direct reinforcement
  • allows us to study influence of structure independent of the structure learning problem (where structure learning is selecting between competing function families)

Experiments

1. Bayesian Optimization Interregation Drilling Game

Design

  • subjects start with 2-6 samples (passive samples)
  • get to choose more samples themselves (active samples)
  • randomly stopped after 2-6 active samples
  • get amount of points equal to highest value point seen so far on screen

Analyses

  1. Subjects sensitive to environment structure
  2. GP params fitted to subject behavior relative to actual (i.e. optimal) parameters that generated the hidden functions
  3. BayesOpt aqusition functions predict better than chance
  4. *got negative response in RLDM reviews - thought not compelling enough
  5. BayesOpt aquisition functions predict behavior better than infomax and heuristics
  6. compare expected improvement (EI) and GP-UCB to infomax (ground-truth counter), as well as POLE, and EXAM (Busemeyer heuristics), perhaps polynomial/spline fits as well?

2. (optional?) Function learning Interregation Drilling Game

Design

  • same as BO drilling game except, at interrogation time, probed for how deep it is at a random spot.
  • amount of points determined by how far away predicted depth is from actual depth

Analyses

  1. Infomax better than BO for explaining active samples - would be nice double dissociation for model fits
  2. * issue: there is a weird disconnect between this, where one has to make a final decision at interrogation time, and experiment 1, where there is no final decision. may be more appropriate to save for Active Decision Making paper
